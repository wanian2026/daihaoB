## 🚀 快速开始

### 1. 启动Web界面

```bash
# 方式一：双击脚本
./start_web.sh

# 方式二：命令行启动
bash start_web.sh

# 方式三：自定义端口
PORT=8080 bash start_web.sh
```

### 2. 打开浏览器

访问：http://localhost:8000

## 💡 使用提示

1. **首次使用**
   - 确保 `config/api_keys.json` 已正确配置
   - 建议先使用模拟交易测试
   - 创建第一个策略配置

2. **风险管理**
   - 使用合理的止损比例
   - 不要使用过高的杠杆
   - 定期检查持仓状态

3. **最佳实践**
   - 从小仓位开始测试
   - 设置合理的参数
   - 定期查看交易历史
