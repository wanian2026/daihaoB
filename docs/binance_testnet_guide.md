# 币安期货测试网 API 密钥获取指南

## 📋 前言

币安期货测试网与正式网是完全独立的系统，**不能使用正式网的API密钥**。必须单独获取测试网的API密钥。

---

## 🚀 快速获取步骤

### 方法1：币安期货测试网（推荐）

#### 1. 访问币安期货测试网
打开浏览器访问：https://testnet.binancefuture.com/

#### 2. 注册/登录测试网账号
- 点击右上角 **"Log In"** 或 **"Register"**
- 使用邮箱注册账号（与正式网账号分开）
- 完成邮箱验证

#### 3. 进入API管理页面
登录后：
1. 点击右上角头像/用户名
2. 选择 **"API Management"** 或 **"API密钥管理"**

#### 4. 创建API密钥
1. 点击 **"Create API"** 按钮
2. 输入Label（标签），例如：`TradingBot` 或 `MyBot`
3. 勾选需要的权限：
   - ✅ **Enable Reading**（读取权限）- 必选
   - ✅ **Enable Futures**（期货权限）- 必选
4. 点击 **"Create"**

#### 5. 保存API密钥（重要！）
创建成功后，系统会显示：
- **API Key**：类似 `abc123def456...`
- **Secret Key**：类似 `xyz789uvw012...`

⚠️ **注意**：Secret Key 只显示一次，请立即复制保存！

### 方法2：OKX模拟交易（备选方案）

如果币安期货测试网无法访问，可以使用OKX的模拟交易：

#### 1. 访问OKX正式网
打开浏览器访问：https://www.okx.com/

#### 2. 注册/登录账号
- 注册OKX账号（可以使用正式网账号）
- 完成实名认证

#### 3. 进入模拟交易
1. 登录后进入"交易"页面
2. 找到并点击"模拟交易"或"Demo Trading"
3. 进入模拟交易环境

#### 4. 创建模拟交易API密钥
1. 在模拟交易环境中，进入API管理
2. 创建API密钥
3. 获取API Key、Secret、Passphrase

## 🔒 API密钥安全建议

对于**正式网**API密钥，请务必：

1. **最小权限原则**
   - ✅ 只勾选 "Enable Reading"（读取）
   - ✅ 只勾选需要的交易权限
   - ❌ **不要勾选 "Enable Withdrawals"（提币）**

2. **绑定IP地址**
   - 在API管理页面设置IP白名单
   - 只允许你的IP地址访问
   - 提高安全性

3. **定期更换**
   - 建议每3-6个月更换一次API密钥

4. **妥善保管**
   - 不要将API密钥提交到代码仓库
   - 不要分享给他人
   - 使用环境变量或加密存储

## ⚠️ 常见问题

### Q1: 为什么不能使用正式网的API密钥？
A: 币安期货测试网和正式网是完全独立的系统，API密钥不通用。必须分别获取。

### Q2: 测试网API密钥能用多久？
A: 测试网API密钥通常长期有效，但如果长期不用可能会失效。

### Q3: 测试资金可以提现到正式网吗？
A: 不可以。测试网的资金是虚拟的，无法提现或转移到正式网。

### Q4: Secret Key显示后还能查看吗？
A: 不能。Secret Key只在创建时显示一次，之后无法查看。如果丢失，需要重新创建API密钥。

### Q5: 币安期货测试网打不开怎么办？
A: 可能是网络问题或测试网维护。可以尝试：
- 检查网络连接
- 使用VPN
- 尝试使用OKX模拟交易作为替代
